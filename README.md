# Portfolio

This is my portfolio website built with Next.js, Tailwind CSS, and TypeScript.

## Features

- Responsive design
- Smooth animations
- Easy to maintain
- SEO friendly

## Technologies Used

- Next.js
- Tailwind CSS
- TypeScript
- Framer Motion
- React Icons

### Jalankan Project 
- npm i
- npm run dev
